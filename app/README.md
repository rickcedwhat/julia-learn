# 🥗 Nutrition Label PWA

This is your workspace for building the Nutrition Label app (React + TypeScript + Vite—not the Learning Portal in `docs/`).

## Getting Started

1. Follow the [Setup Guide](../docs/setup-guide.md) to configure your environment.
2. Open **[Module 2: The Scaffold](../docs/module-2.md)** in the Learning Portal (`npm run docs:dev` from the repo root) and scaffold **this** folder with the copy-paste prompts there.

Your instructor’s curriculum lives in `docs/`; **shippable app code** lives here in `app/`.
